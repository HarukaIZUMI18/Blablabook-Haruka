### Liste des installation pour l'api

* Creation dossier API
```bash
cd api
npm init -y 
npm install express
npm install sequelize
npm install argon2
npm install joi
npm install dotenv
npm install jsonwebtoken 
npm install pg
npm install express-xss-sanitizer
```

#### Dépendence dev
```bash
npm install -D nodemon
```
